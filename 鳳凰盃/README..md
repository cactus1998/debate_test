\# 🏛️ 高中辯論比賽：申論稿與資料表填寫規範



> \\\*\\\*宗旨\\\*\\\*：建立標準化的論證結構與證據格式，確保團隊在場上能夠高效協作、精準擊槌。

\## 📋 一、 申論稿填寫規則 (需、根、解、損)



在政策性辯題中，一份結構完整的申論稿必須包含以下四大核心要素：



\### 1. 需求性 (Necessity / Needs)

\* \*\*核心問項\*\*：現況下存在什麼嚴重的問題？

\* \*\*填寫要求\*\*：

    \* \*\*損害描述\*\*：具體說明受害者、受損範圍或現狀的危機。

    \* \*\*緊急性\*\*：論證為何此問題「現在」就必須被解決，不能再拖延。

    \* \*\*量化與質化\*\*：引用數據說明嚴重程度，並輔以案例增加說服力。



\### 2. 根源性 (Inherency)

\* \*\*核心問項\*\*：為什麼現況下問題無法解決？

\* \*\*填寫要求\*\*：

    \* \*\*結構性成因\*\*：法律漏洞、政策缺失、預算不足等。

    \* \*\*心理性成因\*\*：社會偏見、大眾認知落差、行政怠惰等。

    \* \*\*論證\*\*：必須證明若不透過「改變現狀（變更政策）」，問題將永遠存在。



\### 3. 解決方案 (Solvency)

\* \*\*核心問項\*\*：你的政策建議如何精準解決問題？

\* \*\*填寫要求\*\*：

    \* \*\*具體措施\*\*：包含執行主體、執行手段、所需資源與法源。

    \* \*\*對應邏輯\*\*：方案必須針對「根源性」進行打擊。

    \* \*\*實績證明\*\*：引用專家證言或他國/他地類似經驗，證明方案確實有效。



\### 4. 損益比 (Advantages vs. Disadvantages)

\* \*\*核心問項\*\*：政策帶來的利是否大於弊？

\* \*\*填寫要求\*\*：

    \* \*\*利益 (Pros)\*\*：解決問題後產生的正面價值（如：人權提升、經濟成長）。

    \* \*\*弊害 (Cons)\*\*：實施政策可能帶來的副作用（如：預算赤字、社會動盪）。

    \* \*\*價值判準\*\*：設定一個比較標準（例如：生命權優於財產權），論證利益大於弊害。
